On the SEARCH page you can get the prices for the spare parts, you are looking for

There are to ways to do that:

  • If you want to check a single part number, you can use BY PARTNUMBER tab - just type the part number in the search box
  • If you have a list of part numbers to quote, you can use BY EXCEL FILE tab
    • The process is quite simple:
    • Fill up and save an Excel file in a predefined format with the part numbers, you are looking for
    • You can download the template for this with the TEMPLATE button or use existing one, if you have it already
      • It is mandatory to match the column names to the template
      • Only PARTNUMBER column must be filled, filling of all other columns are optional.
        Filling of the optional columns just clarifies your requirements for the search engine
    • Choose your limit for the lead time in the MAX DAYS box
    • Click UPLOAD button and choose your prepared Excel file
    • You will see the quotation on the screen
    • Now you can fill the QUANTITY field for the positions, you like, and click TO CART to proceed with the order, or click EXPORT button to download the quotation results

На странице OFFERS вы можете скачать файлы с нашими предложениям, разделенные по сроку

Для вашего удобства мы поддеживаем два типа файлов:

  • Сборные файлы, содержащие предложения по всем брендам
  • Файлы по брендам, содержащие предложения только по определенному бренду

Все файлы имеют одинаковую структуру:

  • BRAND - уникальный код бренда (* только для сборных файлов)
  • PARTNUMBER - номер запчасти
  • PRICE (USD) - цена запчасти на нашем складе без доставки в USD
  • PRICE (AED) - цена запчасти на нашем складе без доставки в AED
  • BOOKING - код предложения
  • RATING - статистика поставки
  • QUANTITY – количество на складе (пусто для позиций "под заказ")
  • WEIGHT - физический вес 1 штуки в кг
  • VOLUME - объемный вес 1 штуки в кг
В каждом файле присутствуют отдельные колонки со стоимостью доставки в соответствующую точку (в USD!), например:
  • EKBA (Екатеринбург, РОССИЯ)
  • KIVA (Киев, УКРАИНА)
  • MSKA (Москва, РОССИЯ)

Чтобы рассчитать финальную цену запчасти в вашем регионе, вам нужно сложить цену запчасти без доставки и стоимость доставки до вашего региона
*Если у вас есть ваша индивидуальная скидка, учитывайте, что она применяется только к цене без доставки и не применяется к стоимости доставки

У нас также предоставляется возможность скачивать файлы предложений с нашего FTP-сервера. Для получения доступа, пожалуйста, свяжитесь с нашими менеджерами
